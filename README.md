# ARP Spoofer (C Implementation)

Простая утилита на языке C для выполнения ARP-запросов и спуфинга в локальной сети с использованием Raw Sockets. Проект демонстрирует работу с сетевыми протоколами на низком уровне (L2/L3).

> **Disclaimer:** Данный инструмент создан исключительно в образовательных целях. Использование его в сетях без разрешения владельца является незаконным.

---

## Возможности
*   Формирование произвольных ARP-пакетов вручную.
*   Работа напрямую через сетевой интерфейс в обход стандартного стека TCP/IP.
*   Сканирование сети и перехват трафика (ARP Poisoning).

##  Технологии
*   **Language:** C (C99 standard)
*   **API:** Linux Raw Sockets (`AF_PACKET`)
*   **Tools:** GCC, Make, Ioctl

##  Сборка и запуск

### Системные требования
Для работы требуются заголовочные файлы Linux и права суперпользователя.
```bash
sudo apt-get install build-essential
```

### Компиляция
```bash
gcc main.c -o arpspoofer
```

### Запуск
```bash
sudo ./arpspoofer
```

## Использование
После запуска программа запросит следующие данные:
1.  **Interface**: Имя сетевого интерфейса (например, `eth0` или `wlan0`).
2.  **Target IP**: IP-адрес жертвы.
3.  **Mask IP**: IP-адрес, под который мы маскируемся (например, адрес шлюза).
